## ğŸ« Project Overview
The **ATM Simulation System** is a Java-based project designed to simulate the core functionalities of an Automated Teller Machine (ATM).  
It allows users to:
- Create accounts  
- Log in securely  
- Deposit and withdraw money  
- View mini statements  
- Change their PIN  

This project demonstrates **core Java**, **JDBC**, and **MySQL database connectivity** in a simple yet realistic banking environment.  
It helps students understand object-oriented programming and database interaction in Java-based applications.
